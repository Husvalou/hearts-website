# Heart Resonance

Site web professionnel pour Heart Resonance - Un accompagnement pour se reconnecter à soi.

## Technologies utilisées

- React + TypeScript
- Vite
- CSS moderne

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Le site sera accessible sur `http://localhost:5173`

## Build pour production

```bash
npm run build
```

## Structure du site

- **Maria HUIZAR** : Présentation
- **Séances** : La rencontre de soi
- **Formations** : Programmes structurés
- **Ateliers** : Moments de partage
- **Livres** : Ressources écrites
- **Podcast** : Conversations et réflexions
- **Newsletter** : Inscription à la newsletter

## Style

Design sobre, élégant et professionnel avec beaucoup d'espace blanc, inspiré de sites comme francklopvet.com.
